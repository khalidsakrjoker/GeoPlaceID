GeoPlaceID Software

A professional, easy-to-use desktop tool for extracting latitude and longitude coordinates from Google Maps URLs and Place IDs efficiently without needing API keys.

Available Extraction Methods

📭

Decoding (Fast)

Relies on direct mathematical decoding of the Place ID.
✅ Very Fast ✅ No Internet Required
⚠️ Might miss the target by a few kilometers (provides an area, not an exact point).

🎯

Accurate Scrape (Scraping)

Uses a headless browser (Playwright) to fetch official coordinates from Google.
✅ 100% Accuracy
⏳ Takes longer (~1-3 seconds per place) and requires an internet connection. Auto-saves your progress.

⚖️

Compare Both Methods

Allows you to test both methods together on a limited number of rows to find out the error rate in meters and choose the most suitable one before running the full file.

How to Use

1

Prepare the Excel (CSV) File

Make sure the file contains at least two columns: one named url and the other named Place_ID.

2

Upload File to Software

Open the software, click "Choose File", and select your CSV file. It will show you the confirmed number of rows.

3

Choose Method and Start

Choose the appropriate method (we recommend the second for absolute accuracy), then click "Start Extraction". You can track the progress in the live log at the bottom.
